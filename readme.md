# Automação de API com Cypress

Projeto prático de automação de testes em API REST utilizando Cypress e JavaScript.

## 📌 Objetivo

Praticar automação de requisições HTTP simulando operações CRUD em uma API pública.

---

## 🚀 Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js
- API REST
- JSON

---

## ✅ Cenários automatizados

### GET
- Buscar dispositivo existente
- Buscar dispositivo inexistente
- Buscar dispositivo atualizado
- Validar dispositivo deletado retornando 404

### POST
- Criar novo dispositivo

### PUT
- Atualizar dispositivo criado

### DELETE
- Deletar dispositivo

---

## ✅ Validações realizadas

- Status codes
- Response body
- Cenários positivos e negativos
- Validação pós-delete
